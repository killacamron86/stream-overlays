📋 HOW TO PLAY
🎯
GUESS THE POKEMON
When the game starts type the Pokemon name in chat. First to guess correctly wins the chance to catch it!
⚾
CATCH IT
After guessing correctly throw a ball! Type !pokeball, !greatball, !ultraball or !masterball in chat.
🪙
EARN COINS
Earn coins by chatting (+5 per 2min), guessing (+50), catching (+100), and winning battles (+200).
🏪
SHOP
Spend coins on better balls! Type !shop to see prices, !buy greatball to purchase.
⚔️
BATTLE
Challenge other viewers! Type !battle @username to start a Pokemon battle using your caught Pokemon.
📋
ALL COMMANDS
Type !pokecmds in chat to see every command. Type !pokestats to see your full stats anytime.
